Microsoft Office Suite Office 1.6

Microsoft Office 1.6 was a 16-bit version of the office suite released on July 8, 1991, designed for the Windows 3.0 operating system. It was the last version of Office to support that specific operating system version.

Key Details of Office 1.6

Release Date: July 8, 1991.

Target OS: Windows 3.0.

Architecture: It was a 16-bit application suite.

Successor: It was later superseded by Office 2.0 and Office 2.5 in 1992.

Applications Included

The Microsoft Office 1.6 bundle included the following components:

Word 1.1

Excel 3.0

PowerPoint 2.0

Microsoft Mail 2.1

This version built upon the previous Office 1.5 release by adding the Mail component to the bundle. Like all early Office versions, this is legacy software and no longer supported.
