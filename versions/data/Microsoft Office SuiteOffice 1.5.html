Microsoft Office Suite Office 1.5

Microsoft Office 1.5 was a 16-bit version of the office suite released around 1991, designed for the Windows 3.0 operating system.

Key Details of Office 1.5

Release Date: Circa 1991.

Target OS: Windows 3.0.

Architecture: It was a 16-bit application suite.

Key Update: The primary change from the initial Office 1.0 was the inclusion of Microsoft Excel 3.0, which added features like toolbars and outlining capabilities.

Successor: It was quickly succeeded by Office 1.6 in July 1991, which added Microsoft Mail to the bundle. Applications Included

The Microsoft Office 1.5 bundle included the following components:

Word 1.1

Excel 3.0

PowerPoint 2.0

As with other early versions of Office, this software is now considered legacy and is no longer supported by Microsoft.
