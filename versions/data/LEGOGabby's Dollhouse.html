LEGO Gabby's Dollhouse

Based on the DreamWorks animated series, the LEGO Gabby's Dollhouse theme includes multiple sets featuring characters and locations from the show. The sets are part of the LEGO 4+ line, designed for young builders with features like Starter Bricks and simple instructions.

Current and upcoming LEGO Gabby's Dollhouse sets

These sets are either available now or confirmed for release in 2025.

LEGO Gabby's Dollhouse Mermaid Gabby's Aquarium Adventure (11204): Recreate underwater scenes with Gabby, dressed as a mermaid, and MerCat. The 103-piece set includes a multi-colored jellyfish and other sea creatures for imaginative play.

LEGO Gabby's Dollhouse Sweet Treat Mountain & Kitty Garden (11205): With 263 pieces, this set features locations from the 2025 movie along with Gabby, Pandy Paws, Chumsley, and other characters. Builders can explore a candy-colored mountain with donut floats and ice cream cones.

LEGO Gabby's Dollhouse Gabby's Kitty Care Ear (10796): In this 165-piece set, build a room shaped like a cat's ear. It includes a tiny baby kittycorn, a cat-owl (me-owl), and accessories for imaginative pet care.

LEGO Gabby's Dollhouse Gabby's Party Room (10797): This 252-piece set features a party room and a pink car. It includes a buildable DJ booth and dance floor.

LEGO Gabby's Dollhouse Crafting with Baby Box (10795): Join Gabby and Baby Box Cat in this 60-piece set to create their own model of the dollhouse. Includes a paintbrush, paint, and scissors accessories.

Previously released LEGO Gabby's Dollhouse sets

These sets were released in 2023 but may still be available from various retailers.

LEGO Gabby's Dollhouse (10788): This large, 498-piece set is an 8-room playhouse featuring a sliding elevator connecting the different floors. It includes Gabby, Pandy Paws, MerCat, and Cakey figures, along with many accessories for baking, making music, and more.

LEGO Gabby's Dollhouse Kitty Fairy's Garden Party (10787): A 130-piece set for staging a garden party. It includes a treehouse, a swing, and Pandy Paws and Kitty Fairy figures.

LEGO Gabby's Dollhouse Gabby & MerCat's Ship & Spa (10786): With 88 pieces, this set features a buildable boat and spa for Gabby and MerCat.

LEGO Gabby's Dollhouse Bakey with Cakey Fun (10785): A 58-piece set centered around baking with Cakey Cat. It includes a small buildable kitchen and a giant cupcake.
