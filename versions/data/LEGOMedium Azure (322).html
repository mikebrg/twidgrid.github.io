LEGO Medium Azure (322)

LEGO Medium Azure (322) is a bright, solid light-blue color introduced in 2010 that is widely used across many LEGO themes, particularly those aimed at younger audiences or requiring vibrant color palettes like LEGO Friends and LEGO Disney Princess.

Overview and Usage

Color Profile: Medium Azure (officially "Medium Azur") is a cheerful, light blue shade with the Element ID 322. It is often used in conjunction with transparent light blue elements in sets to represent water or other transparent effects.

Availability: The color is available in many standard brick, plate, tile, and specialized elements across the LEGO portfolio.

Key Themes: It was first widely announced and used in the LEGO Friends theme, where it contributed to the bright aesthetic of Heartlake City buildings and vehicles. It also appears heavily in:

LEGO Disney Princess: Used for various castle and underwater builds, such as Ariel's Underwater Palace.

LEGO Classic: Frequently included in large creative boxes and supplemental sets for general building and color sorting activities.

LEGO Creator: Used as an accent color in some Creator 3-in-1 sets, providing an appealing and modern look.

LEGO Ninjago: Featured in detailed sets like the Ninjago City Gardens as an accent color for water features.

Examples of Sets with Medium Azure

Medium Azure is a versatile color used to add vibrant detail to many different models:

Olivia's House (3315): One of the early sets in the Friends theme to prominently feature the color in its design.

Maersk Line Triple-E (10241): This highly detailed adult collector's ship model used Medium Azure as a primary color for the hull and containers.

Ninjago City Gardens (71741): Multiple elements in this set, including water features and architectural details, use the Medium Azure color.

Townhouse Toy Store (31105): A Creator 3-in-1 set that used Medium Azure for the second-floor facade, a design choice praised for making the color more widely available in specific parts.
