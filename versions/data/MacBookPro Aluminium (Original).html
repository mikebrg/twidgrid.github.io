MacBook Pro Aluminium (Original)

The original aluminum MacBook Pro model, released on February 14, 2006, featured an Intel Core Duo processor, a 15.4-inch display, up to 2GB of RAM, and introduced key innovations like the MagSafe power connector and a backlit keyboard.

Key Features and Ports

MagSafe Power Connector: A new magnetic power connector designed to safely disconnect if the cord was accidentally tugged.

Built-in iSight Camera: An integrated webcam for video conferencing.

Backlit Keyboard: Features an ambient light sensor that adjusts keyboard illumination and screen brightness.

Ports: Includes two USB 2.0 ports, one FireWire 400 port (dropped the FireWire 800 port from its predecessor), a Gigabit Ethernet port, and an ExpressCard/34 expansion slot (replacing the PC card slot).

Software Support: The maximum supported operating system for this Core Duo model is Mac OS X 10.6.8 "Snow Leopard". It cannot run OS X 10.7 Lion or later versions due to its 32-bit CPU.

Design: Encased in an aluminum enclosure, it weighs 5.6 pounds (2.54 kg) and is one inch thick.
