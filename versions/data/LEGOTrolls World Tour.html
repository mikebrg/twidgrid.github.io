LEGO Trolls World Tour

The LEGO Trolls World Tour theme was released in 2020 and discontinued in July 2021. Because the sets are retired, they are no longer available from official LEGO retailers but may be found on secondary markets like eBay or Amazon. The theme is based on the movie of the same name and includes minifigures of characters like Poppy, Branch, and Queen Barb.

Collectible and smaller sets

These smaller sets are a good way to start collecting the unique Trolls minifigures and include some 4+ sets designed for younger builders.

LEGO Trolls World Tour Poppy's Pod (41251): This set lets you build Poppy's pod-shaped house, complete with furniture and accessories. It includes a Poppy minifigure and was designed to be combined with the Pop Village Celebration set.

LEGO Trolls World Tour Techno Reef Dance Party (41250): A 159-piece set that features a dance party scene from the movie, including a DJ booth and a Mermaid minifigure.

LEGO Trolls World Tour Lonesome Flats Raft Adventure (41253): This set lets you build a musical raft for Poppy, Branch, and Hickory to float down the river. It's designed as a 4+ set and includes a spinning dancefloor.

LEGO Trolls World Tour Poppy's Carriage (30555): A promotional polybag set that includes Poppy and her insect-powered carriage. Due to its limited release, it is a harder-to-find collectible.

Larger stage and village sets

These bigger sets feature more pieces and characters, allowing for more elaborate music scene recreations.

LEGO Trolls World Tour Pop Village Celebration (41255): With 380 pieces, this set builds a treehouse-style pop village. It comes with Poppy, Branch, Cooper, Guy Diamond, and Tiny Diamond minifigures, along with two portable pods that can be hung from the tree.

LEGO Trolls World Tour Vibe City Concert (41258): This 494-piece concert stage set includes a recording room, DJ booth, and musical instruments. It comes with Poppy, Branch, Cooper, Funk Troll (Female), and Hickory minifigures.

LEGO Trolls World Tour Volcano Rock City Concert (41254): In this 387-piece set, you can recreate the climactic rock concert with minifigures of Poppy, Branch, and Queen Barb. The stage includes guitars and microphones for the show.

Vehicle sets

LEGO Trolls World Tour Rainbow Caterbus (41256): This 395-piece set builds the colorful caterbus and a bus stop. It includes minifigures of Poppy, Cooper, and Cloud Guy, along with three cloud babies.

LEGO Trolls World Tour Poppy's Air Balloon Adventure (41252): A 250-piece set featuring a pink hot air balloon. The set includes minifigures of Poppy, Branch, and Biggie with Mr. Dinkles.
