LEGO Chess

The LEGO Chess Video Game (Discontinued)

Released in 1998 for Microsoft Windows, LEGO Chess was a strategy game developed by Krisalis Software.

Gameplay: The game adhered to the traditional rules of chess but featured humorous, animated cutscenes when a piece was captured, depending on the chosen theme. Players could choose between a Western or Pirates theme in "Story Mode" or mix them in "Quick Match" (though mixing disabled the animations).

Tutorial: The game included a comprehensive tutorial mode guided by an "Elvis Presley-esque" voiced King minifigure.

Availability: The game is discontinued and no longer officially sold on digital platforms. Physical copies can be found on re-seller sites like eBay, but the game has compatibility issues with modern operating systems and requires fan-made patches to run.

Physical LEGO Chess Sets (Current & Discontinued)

The LEGO Group has produced numerous physical, brick-built chess sets over the years, many of which are now retired collector's items.

Traditional Chess Set (40719) Status: Currently available for purchase from the official LEGO website and retailers.

Description: Released in 2024, this set features a classic, elegant design with a dark brown and beige board and traditional, stud-free pieces. It also includes pieces for playing checkers and can split in two for easy storage.

Retired Sets: Past sets included various themes and styles, which are now only available from re-sellers:

Knights' Kingdom Chess Set (851499): The first official set, released in 2005, it included 24 minifigures from the Knights' Kingdom II storyline.

Hogwarts Wizard's Chess Set (76392): Released in 2021 as a tie-in with the Harry Potter series, it recreates the iconic chess challenge scene from the film.

Iconic Chess Set (40174): This set, retired in 2022, featured a more blocky, traditional design with no minifigures, allowing pieces to be stored inside the board.

Pirates Chess Set (40158): A popular set with a beach theme that was divided between a Royal Navy faction and a pirate crew.
