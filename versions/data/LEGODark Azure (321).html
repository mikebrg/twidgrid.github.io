LEGO Dark Azure (321)

LEGO Dark Azure (321) is a rich, solid blue color introduced in 2011 and is a common color found across a wide variety of LEGO themes, most notably in the LEGO Speed Champions line for representing specific car liveries.

Overview and Usage

Color Profile: Dark Azure is a deep, vibrant blue, officially known by LEGO as "Dark Azur" and with the Element ID 321.

Availability: It is widely available across a multitude of sets and part types, including basic bricks, plates, slopes, and specialized elements.

Key Themes: While not exclusive to any single theme, Dark Azure is notably used in:

LEGO Speed Champions: It is the specific "Bugatti Blue" used for models like the Type 35. The upcoming 2025 sets like the Lamborghini Revuelto & Lamborghini Huracán STO (77238) use the color.

LEGO Creator: Appears in various Creator 3-in-1 sets that offer build and rebuild options for different models.

LEGO Friends and LEGO Disney Princess: Often used for accent details in buildings or vehicles, contributing to the bright color palettes of these themes.

LEGO Classic: Included in basic brick sets to provide builders with a versatile color palette for free-building.

Examples of Sets with Dark Azure

Dark Azure is a versatile color used to add vibrant detail to many different models.

Lamborghini Revuelto & Lamborghini Huracán STO (77238): Features the color in various parts of the car models.

The Simpsons House (71006): Not a primary color, but small pieces in this color may appear as details within the set.

Mighty Dinosaurs (31058): The Creator line often uses a broad spectrum of colors, including Dark Azure, in its varied pieces.
