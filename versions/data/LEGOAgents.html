LEGO Agents

The Lego Agents theme is a discontinued product line that ran from 2008 to 2009, with a successor theme, Lego Ultra Agents, available from 2014 to 2015. These sets are no longer in production but may be found through re-sellers.

Original Lego Agents Theme (2008-2009) Inspired by James Bond and other spy movies, this theme featured an elite team of secret agents battling the international crime syndicate led by the evil Dr. Inferno. The sets focused on action-packed missions in various locations.

Mobile Command Center (Set number: 8635): The agents' central hub vehicle for coordinating missions.

Volcano Base (Set number: 8637): Dr. Inferno's main hideout from the second wave of sets.

Robo Attack (Set number: 8970): A large, poseable robot used by Dr. Inferno in the Agents 2.0 storyline. Aerial Defense Unit (Set number: 8971): A large helicopter for the agents, part of the final wave.

Lego Ultra Agents Theme (2014-2015)

The "Ultra Agents" theme was an update to the original, featuring an updated team of agents (e.g., Agent Jack Fury, Agent Caila Phoenix) fighting a new group of villains led by AntiMatter in Astor City. These sets were designed to be used with a companion app for extra digital features.

Ultra Agents Mission HQ (Set number: 70165): The primary mobile headquarters for the Ultra Agents team.

Toxikita's Toxic Meltdown (Set number: 70163): Featured the villainess Toxikita and her toxic-spilling vehicle.

Hurricane Heist (Set number: 70164): Included a high-tech vehicle to stop the villain Psyclone.

UltraCopter vs. AntiMatter (Set number: 70170): A major set from the second wave featuring the main antagonist, AntiMatter.
