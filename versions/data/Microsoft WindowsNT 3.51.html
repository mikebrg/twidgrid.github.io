Microsoft Windows NT 3.51

Windows NT 3.51 is the third major release of the Windows NT operating system, developed by Microsoft for the data server and personal workstation markets. It was released on May 30, 1995, eight months following the release of Windows NT 3.5. The most significant enhancement offered in this release was that it provides client/server support for inter-operating with Windows 95, which was released almost three months after NT 3.51. Windows NT 4.0 became its successor a year later. Mainstream support for Windows NT 3.51 Workstation ended on December 31, 2000, and extended support ended on December 31, 2001, while Windows NT 3.51 Server mainstream support ended on September 30, 2000, followed by extended support on September 30, 2002. Both editions were succeeded by Windows NT 4.0 Workstation and Windows NT 4.0 Server, respectively.
