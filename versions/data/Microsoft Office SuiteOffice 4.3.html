Microsoft Office Suite Office 4.3

Microsoft Office 4.3 was released on June 2, 1994, as the final 16-bit version of the suite, primarily designed for the Windows 3.1x operating system.

Key Details of Office 4.3

Release Date: June 2, 1994.

Target OS: Windows 3.1x, Windows NT 3.1, and Windows NT 3.5. Architecture: It was a 16-bit application suite. A separate, partially 32-bit version called "Office for NT 4.2" was released for Windows NT platforms.

Editions: It was available as a Professional Edition that included a comprehensive set of applications.

Components of Office 4.3 Professional

The Professional Edition of Office 4.3 included the following applications:

Word 6.0

Excel 5.0

PowerPoint 4.0

Access 2.0

Mail 3.2

Some versions also came bundled with Microsoft Bookshelf.

Legacy Status

Office 4.3 was eventually superseded by Microsoft Office 95, which was the first fully 32-bit version of Office. Like other legacy software, Office 4.3 is no longer supported by Microsoft. You can find historical copies preserved for educational purposes on resources like the Internet Archive.
