LEGO Duplo Winnie The Pooh

The LEGO Duplo Winnie the Pooh theme has appeared in the LEGO Duplo product line across three different periods: an original run from 1999 to 2001, a brief return in 2011, and a reintroduction in 2025. The sets feature large, safe Duplo bricks and figures of the characters for young children (ages 1.5-5).

Overview

This theme was the first ever licensed Duplo theme and focused on the residents of the Hundred Acre Wood. The sets are designed to help toddlers with early development skills, such as color-sorting, counting, and exploring emotions through imaginative play. The sets usually depict familiar locations like Pooh's house or a picnic area.

Characters Included

The Duplo theme includes figures of the core characters from the Disney version of the franchise:

Winnie the Pooh

Tigger

Piglet

Eeyore

Set Examples

The sets vary in size and typically include a few characters and simple builds:

Winnie the Pooh's House (5947 / 2981): Multiple versions of Pooh's home, "Mr. Sanders," have been released across the different waves.

The Hundred Acre Wood (2987): A larger set from the original 1999 run that included figures for Pooh, Piglet, Tigger, and Eeyore.

Winnie the Pooh's Birthday Party (10457): A currently available set (as of 2025) that includes Winnie the Pooh, Tigger, and Piglet figures with double-sided faces to help toddlers explore different emotions.

Tigger's Expedition (5946): A set from the 2011 wave that was essential for getting a Tigger figure during that specific release period.

Availability

Older sets from the 1999-2001 and 2011 runs are retired and only available through secondary marketplaces like eBay. The newest set, Winnie the Pooh's Birthday Party (10457), is a current, active product available from official LEGO retailers as of 2025.
