LEGO Castle

The LEGO Castle theme is a classic LEGO line with a rich history dating back to 1978. While there is no current, dedicated "Castle" theme with ongoing multiple waves of sets, LEGO consistently releases standalone castle-themed products and licensed sets that cater to fans of medieval building.

History of the Theme

The Castle theme was one of the original three LEGOLAND themes, evolving from simple yellow castles into a complex world of various knightly factions, outlaws, and fantasy creatures. Over the decades, it has featured numerous sub-themes:

Classic Castle (1978-1983): The iconic "Yellow Castle" that launched the theme.

Factions Era (1984-1999): A "golden period" with competing groups like the Black Falcons, Lion Knights/Crusaders, Forestmen, and Dragon Masters.

Fantasy Eras (2000s): Themes like Knights' Kingdom II and the 2007 Castle theme introduced more fantasy elements like wizards, skeletons, trolls, and orcs.

Kingdoms (2010-2012): A return to a more realistic medieval setting with a war between the Lion Kingdom and Dragon Kingdom.

Current and Available "Castle" Sets (as of December 2025)

LEGO no longer uses the general "Castle" theme name for ongoing lines, but it has released several highly popular sets under different themes, often as homages to the classic era:

Lion Knights' Castle (10305):

Description: This large, detailed adult collector set (4,514 pieces) was released for the LEGO Group's 90th anniversary in 2022. It is a modern homage to the classic castle themes, featuring a working drawbridge, water wheel, a queen minifigure, and an homage to the classic Forestmen faction.

Status: This set is rumored to be retiring at the end of 2025 and is currently out of stock on the official LEGO shop.

Medieval Market Village (10332):

Description: Released in March 2024 as a remake of the beloved 2009 set, this set features various medieval-era buildings like a tavern, a cheese shop, and a blacksmith. It is a highly detailed set for adult fans and includes a sought-after minifigure goat piece.

Status: This set is currently available from the official LEGO shop.

Creator 3-in-1 Medieval Horse Knight Castle (31168):

Description: This set is part of the versatile Creator 3-in-1 line, which means it can be rebuilt into two alternate models: a castle tower with a catapult or a marketplace with a water mill. It includes new minifigure factions: the Horse Knights and the Serpent Knights.

Status: This set is currently available for purchase.

LEGO Ideas Dungeons & Dragons: Red Dragon's Tale (21348):

Description: A recent licensed set that incorporates many fantasy castle elements, including a large brick-built red dragon, a tavern, a dungeon, and six unique minifigures.

Status: This set is an exclusive and is currently available for purchase.
