PDF Editor

Here’s what it does out of the box:
- Upload an existing PDF or start a blank doc.
- Render pages with PDF.js.
- Overlay edits with Fabric.js: select/move, freehand pen, text boxes, rectangles, image stamps, draggable highlights.
- Undo/redo, delete, zoom in/out, fit-to-screen, page navigation with thumbnails.
- Export to a flattened, downloadable PDF using pdf-lib (keeps the original page content + your overlay on the current page).

How to use it:
- Save the HTML file from the canvas and open it — it works locally (no server required).
- Drop it into any page or wrap it as an iframe “widget”.
- Optional: restyle the toolbar to match your brand (it’s just CSS).
