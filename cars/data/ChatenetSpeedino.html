Chatenet Speedino

The Speedino is one of Chatenet’s sporty / sporty-look microcars. It attempts a more “fun” style compared to some ultra utilitarian models, while still fitting in the light quadricycle / licence-exempt or licence-minimal classification in various countries.

Engine
Petrol (2-cylinder inline), ~523 cc, ~15 kW = ~20 hp in certain petrol versions.
Diesel version - There was also a Lombardini diesel version in later years.
Transmission Automatic / CVT in many spec sheets.
Body / Seats 2 seats, 2-door roadster / cabriolet style in some trims. Some versions may have a more closed / coupe-like roof.
Maximum speed - Early petrol version ~110 km/h.
Weight / dimensions - One data point: ~395 kg in petrol variant (2004), length ~2.95 m, width ~1.555 m, height ~1.25 m.
